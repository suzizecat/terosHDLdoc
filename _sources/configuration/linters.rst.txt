.. _linters:

Linters
=======

Verilog/SV linter selection
---------------------------

In the plugin configuration go to ``Teroshdl › Linter › Verilog › Linter: A`` and select a
linter that you have installed in your system.

.. image:: images/verilog_linter.png

Configure the linter
~~~~~~~~~~~~~~~~~~~~

.. image:: images/verilog_linter_settings.png

Configure  the settings of the linters you are using.

-  **Arguments**:
    Add custom arguments to the linter
-  **Path**:
    If the linter is not in the system path you have to configure the path
-  **Bin**:
    TBD
-  **Enable**:
    Enable the custom call configured in the ``Bin`` section
