.. _error_checking:

Errors checking
===============

With a simulator installed in the system is possible to have linter funtionalities.
The errors are underlined in the opened files. It is necesary to save the file to update the linter results.

The list of errors in the file appears in the console. It is also possible to use it to navigate between the errors.
Hovering the error will show the simualtor error message.

.. image:: images/errors_check.png

There are more information about supported linters in the :doc:`installation <../installing>` section.
And more information about the :doc:`linters configuration <../configuration/linters>` in its section.
