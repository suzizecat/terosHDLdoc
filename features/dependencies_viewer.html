Dependencies viewer¶

The dependencies viewer can create a dependencie graph of the given files. Aditionally it’s possible to export the documentation of all the provided files along with the dependencies graph and an index linking all the generated documentaion files.

Important

Requirements!

To be able to use this functionality is necesary to have Python3 configured in TerosHDL More info in the configuration section

Usage instructions¶

  1. Open the command palette Ctrl+Shift+P and select Open dependencies viewer

  1. Add a HDL files to the viewer (you can mix verilog and VHDL).
